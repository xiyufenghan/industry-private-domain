# 宝塔面板部署指南（纯图形界面，零命令行）

## 第一步：安装必要软件

打开宝塔面板，点左侧菜单「**软件商店**」：

1. 搜索「**Node.js版本管理器**」→ 点「安装」→ 安装完成后点「设置」→ 选择安装 **Node 18** 版本
2. 搜索「**PM2管理器**」→ 点「安装」

## 第二步：上传项目文件

1. 点左侧菜单「**文件**」
2. 进入 `/www/wwwroot/` 目录
3. 点上方「**上传**」按钮，把整个项目压缩成 **zip** 上传
4. 上传完成后，右键 zip 文件 → 「**解压**」
5. 确认解压后目录结构为 `/www/wwwroot/private-domain/`，里面有 `client/`、`server/`、`package.json` 等

> 💡 **打包提示**：在你电脑上，把「行业私域工作间」文件夹里的所有内容（不含 `node_modules` 文件夹）压缩成 zip，重命名文件夹为 `private-domain`

## 第三步：用宝塔终端执行安装（只需粘贴一次）

1. 点左侧菜单「**终端**」（宝塔自带的网页终端，不需要额外软件）
2. 把下面这段**复制粘贴**进去，按回车，等它跑完就行：

```
cd /www/wwwroot/private-domain && cd client && npm install && npm run build && cd ../server && npm install && npx prisma generate && npx prisma db push && npx tsx prisma/seed.ts && echo "✅ 全部完成！"
```

> 看到 `✅ 全部完成！` 就说明成功了。这一步大概等 2-3 分钟。

## 第四步：PM2 管理器启动项目

1. 点左侧菜单「**软件商店**」→ 找到「PM2管理器」→ 点「**设置**」
2. 点「**添加项目**」
3. 填写：
   - **启动文件**：点右侧文件夹图标，选择 `/www/wwwroot/private-domain/ecosystem.config.js`
   - **项目名称**：会自动填，或手动写 `private-domain`
4. 点「**提交**」
5. 看到状态变成绿色「**运行中**」就成功了 ✅

## 第五步：配置域名访问（可选）

**不配域名**：直接浏览器访问 `http://你的服务器IP:3001` 就能用。

**配域名**（更专业）：

1. 点左侧「**网站**」→ 「**添加站点**」
2. 填写：
   - **域名**：填你的域名，比如 `workspace.example.com`
   - **PHP版本**：选「纯静态」
   - **数据库**：不创建
3. 点「提交」
4. 点刚创建的站点名称 → 左侧选「**反向代理**」
5. 点「**添加反向代理**」：
   - **代理名称**：随便写，如 `app`
   - **目标URL**：填 `http://127.0.0.1:3001`
   - **发送域名**：填 `$host`
6. 点「提交」

现在访问你的域名就能看到登录页面了！

## 第六步：SSL 证书（可选，让网址变成 https）

1. 站点设置 → 左侧「**SSL**」→ 「**Let's Encrypt**」
2. 勾选域名 → 点「申请」
3. 申请成功后开启「**强制HTTPS**」

## 以后更新代码

1. 本地改好代码后，重新压缩 zip 上传覆盖
2. 宝塔「终端」粘贴：`cd /www/wwwroot/private-domain/client && npm run build`
3. PM2 管理器里点「重启」按钮

## 备份数据

数据库文件在 `/www/wwwroot/private-domain/server/dev.db`

在宝塔「文件」里找到这个文件，右键「**下载**」就是备份了。建议每周下载一次。
